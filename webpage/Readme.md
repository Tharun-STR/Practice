# 🚀 Coming Soon Website

A modern, responsive **Coming Soon / Under Maintenance** webpage built using **HTML, CSS, and JavaScript**. This project includes a countdown timer, animations, and a clean UI design suitable for launching products or websites.

---

## ✨ Features

* 📱 Fully responsive design (mobile, tablet, desktop)
* ⏳ Live countdown timer
* 🎨 Modern UI with background image overlay
* 🚀 Animated rocket illustration
* 🖱️ Interactive button with hover effects
* 🔤 Clean typography using Google Fonts (Poppins)

---

## 📂 Project Structure

```
project-folder/
│── index.html
│── style.css
|── rocket.png
```

---

## ⚙️ How It Works

* The countdown timer is implemented using JavaScript.
* It calculates the difference between the current time and the launch date.
* Updates every second using `setInterval()`.

---

## 🛠️ Setup Instructions

1. Clone or download the project
2. Open the folder
3. Run `index.html` in your browser

No installation or dependencies required.

---

## 🎯 Customization

### Change Launch Date

Inside `script`:

```js
var countDownDate = new Date("Apr 25, 2026 00:00:00").getTime();
```

Update the date as needed.

---

### Change Images

Replace image URLs or local files:

* Logo
* Background image
* Rocket image

---

### Colors & Styling

Modify in CSS:

* Primary color: `#ff4b2b`
* Background overlay opacity
* Font sizes
